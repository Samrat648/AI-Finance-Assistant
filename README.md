# 💰 AI Finance Assistant

An AI-powered Finance Assistant built with Flask and Ollama (Gemma 3). It helps users understand budgeting, savings, and general financial topics.

## Features

- 💰 Budget Planning
- 📊 Expense Analysis
- 💵 Savings Suggestions
- 🤖 AI Financial Guidance
- 🔒 Local AI with Ollama
- ⚡ Fast & Lightweight

## Tech Stack

- Python
- Flask
- Ollama
- Gemma 3

## Installation

```bash
pip install -r requirements.txt
```

Start Ollama

```bash
ollama serve
```

Run the project

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```
